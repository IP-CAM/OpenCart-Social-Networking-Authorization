VERSION 2.1
UA
In version 2.1, the ability to configure authorization via Instagram has been added. The authorization algorithm via Facebook has also been changed - it now works via OAuth 2.

Installing the module
1. Upload the module files to the root of the project
2. Install the 'WS SocialAuth' module in Add-ons -> Modules
3. Use the instructions on the settings page of the 'Install' module to make additional settings.

Facebook API settings
Create a Facebook application https://developers.facebook.com/apps/
Make the application public and register the callback href (take in the module settings)
Prescribe App ID and Secret App in the module settings

Google+ API settings
Create a Google app https://console.cloud.google.com/apis/
Prescribe callback href (take in module settings)
Prescribe the App ID and secret key in the module settings

Instagram API settings
Register developer - https://www.instagram.com/developer/register/
Create Client - https://www.instagram.com/developer/clients/register/
Prescribe callback href (take in module settings)
Prescribe the App ID and secret key in the module settings

Налаштування API Google+
Створити додаток Google https://console.cloud.google.com/apis/
Прописати  callback href  (взяти у налаштуваннях модуля)

Зареєструвати розробника - https://www.instagram.com/developer/register/
Створити Client - https://www.instagram.com/developer/clients/register/
Прописати  callback href  (взяти у налаштуваннях модуля)
